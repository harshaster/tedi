# tedi

just a TExt EDitor.

## Features

- Just start writing.
- Increase or Descrease font
- Change font colour
- Save text file
- Open text file
- Sync changes
- Get it back anytime you open
- Now supports markdown (update: 14-7-2025)
- Mermaid diagrams are also supported (update 31-5-2026)
